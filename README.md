# areyoualive
基于HomeAssistant和某位大神的app-reporter做的窗口实践小工具（不是
# 如何食用
1.下载index.php<br>
2.下载 <a href="https://github.com/maoawa/sleepy-helper">sleep-helper</a> 中的app-reporter<br>
3.根据app-reporter设置好自己的窗口上传<br>
4.在php文件里的$CONFIG内以同样方式设置<br>
5.php -S 0.0.0.0:8080  即可食用<br>
